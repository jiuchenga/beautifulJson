## What is JSON?

JSON (JavaScript Object Notation) is a lightweight data-interchange format that is easy for humans to read and write and easy for machines to parse and generate. It's the most common format for APIs and configuration files.

## Why Format JSON?

Raw JSON from APIs is often minified (single line) or poorly formatted. A JSON formatter helps you:

- **Read and understand** complex JSON structures
- **Debug** API responses
- **Validate** JSON syntax
- **Compare** different JSON documents

## Common Use Cases

### API Development

When building or consuming APIs, you'll often need to format JSON responses for debugging:

```json
{
  "status": "success",
  "data": {
    "users": [
      { "id": 1, "name": "Alice" },
      { "id": 2, "name": "Bob" }
    ]
  }
}
```

### Configuration Files

Many tools use JSON configuration (tsconfig.json, package.json, .prettierrc). Our formatter helps keep them clean and readable.
